# 🧬 Machine Learning Based – Multi-Omics Analysis for Breast Cancer (BRCA)

---

## 📦 Setup (Local)

### 1. Clone the repo
```
git clone https://github.com/xkuubix/ml-omics.git
cd ml-omics
```
### 2. Install uv and python (if you don't have it)
```
pip install uv
uv python install 3.11
```
### 3. Create virtual environment and install dependencies
```
uv sync
```

### 4. Open&run notebook in VS-code, or in jupyter lab
```
uv run jupyter notebook
```

## ☁️ No setup (Google Colab)
